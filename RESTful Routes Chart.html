RESTful Routes

A table of all 7 RESTful routes for the blog app

Name Path HTTP Verb Purpose Mongoose Method
Index /blogs GET List all blogs Blog.find()
New /blogs/new GET Show new blog form N/A
Create /blogs POST Create new blog, then redirect somewhere Blog.create()
Show /blogs/:id GET Show info about the specified blog Blog.findById()
Edit /blogs/:id/edit GET Show the edit form for specified blog Blog.findById()
Update /blogs/:id PUT Updates record for the specified blog Blog.findByIdAndUpdate()
Destroy /blogs/:id DELETE Delete the specified blog, then redirect somewhere Blog.findByIdAndRemove()
